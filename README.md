# DSA Progress Dashboard — GitHub Pages Edition

## What this is
- **`index.html`** → Your dashboard (static), hosted on GitHub Pages
- **`data.json`** → Generated progress data
- **GitHub Actions** → Automatically fetches from Notion every 10 minutes and updates `data.json`

No Vercel needed — everything stays on GitHub with automatic syncing!

---

## Setup Instructions

### Step 1 — Get Your Notion Token

1. Go to https://www.notion.so/my-integrations
2. Click **New integration**
3. Give it a name, select your workspace
4. Copy the **Internal Integration Token** (`secret_...`)
5. In your Notion DSA page → click `...` → **Connections** → add your integration

### Step 2 — Create GitHub Secrets

1. Go to your repo → **Settings → Secrets and variables → Actions**
2. Click **New repository secret** and add:
   - **Name:** `NOTION_TOKEN` → **Value:** (paste your token from Step 1)
   - **Name:** `PAGE_ID` → **Value:** `363bc09fccb2812aab8ac19b355f59fa` (or your Notion page ID)

### Step 3 — Enable GitHub Pages

1. Go to repo **Settings → Pages**
2. Set **Source** to: `Deploy from a branch`
3. Select **Branch:** `main` / **Folder:** `/ (root)`
4. Save
5. Your dashboard will be at: `https://yourusername.github.io/your-repo/`

### Step 4 — Activate GitHub Actions

1. Go to **Actions** tab in your repo
2. Click **Sync Notion Data** workflow
3. Click **Enable workflow** (if disabled)
4. The workflow will run automatically every 10 minutes
5. You can also manually trigger it: **Run workflow**

## How It Works

```
GitHub Actions (every 10 min)
  └─► fetch-notion-data.js
       └─► Notion API (counts your toggles)
            └─► writes to data.json
                 └─► commits & pushes to repo

Browser (GitHub Pages)
  └─► fetches data.json (no CORS issues!)
       └─► displays updated progress
```

### Local Testing

To test the Notion fetch locally:
```bash
export NOTION_TOKEN="secret_xxxx"
export PAGE_ID="363bc09fccb2812aab8ac19b355f59fa"
npm run fetch
```

## Customizing Targets

Each section has a **target** input in the dashboard. Edit any target and it saves to your browser's localStorage — persists across refreshes and syncs.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| **GitHub Actions failing** | Check repo → Actions tab → see the error. Ensure `NOTION_TOKEN` is set in Secrets. |
| **`data.json` not updating** | Manually trigger the workflow: Actions → Sync Notion Data → Run workflow |
| **Dashboard shows error** | Open browser DevTools (F12) → Console → check for fetch errors. Ensure `data.json` exists. |
| **Notion token expired** | Go to integrations at notion.so → regenerate token → update the GitHub secret |
