# 🚀 Complete Deployment Guide

## Step 1: Initialize Git & Commit

```bash
cd /Users/naveed/Downloads/DSA_STATUS

# Initialize Git repo
git init

# Add remote (replace USERNAME and REPO_NAME)
git remote add origin https://github.com/USERNAME/REPO_NAME.git

# Add all files
git add .

# First commit
git commit -m "initial: DSA progress dashboard with GitHub Actions sync"

# Push to GitHub (assumes main branch exists on GitHub)
git branch -M main
git push -u origin main
```

---

## Step 2: Create GitHub Secrets

1. Go to: `https://github.com/USERNAME/REPO_NAME/settings/secrets/actions`
2. Click **New repository secret**
3. Add these two secrets:

### Secret 1: NOTION_TOKEN
- **Name:** `NOTION_TOKEN`
- **Value:** `secret_xxxxxxxxxxxxx` (from https://www.notion.so/my-integrations)

### Secret 2: PAGE_ID
- **Name:** `PAGE_ID`
- **Value:** `363bc09fccb2812aab8ac19b355f59fa` (or your Notion page ID)

---

## Step 3: Get Your Notion Token

1. Go to https://www.notion.so/my-integrations
2. Click **+ New integration**
3. Fill in the name (e.g., "DSA Dashboard")
4. Select your workspace
5. Click **Submit**
6. Copy the **Internal Integration Token** (starts with `secret_`)
7. In your **Notion DSA page** → click `...` → **Connections** → add your integration

---

## Step 4: Enable GitHub Pages

1. Go to: `https://github.com/USERNAME/REPO_NAME/settings/pages`
2. Under **Source**, select:
   - **Deploy from a branch**
   - **Branch:** `main`
   - **Folder:** `/ (root)`
3. Click **Save**
4. Wait ~1 minute for deployment

Your site will be live at: **`https://USERNAME.github.io/REPO_NAME/`**

---

## Step 5: Verify GitHub Actions is Running

1. Go to: `https://github.com/USERNAME/REPO_NAME/actions`
2. Look for **Sync Notion Data** workflow
3. If it's disabled, click **Enable workflow**
4. The workflow runs automatically every 10 minutes
5. Or manually trigger: **Run workflow** button

## 🐛 Troubleshooting

| Problem | Solution |
|---------|----------|
| **Workflow not running** | Go to Actions tab → Sync Notion Data → Enable workflow |
| **`data.json` not updating** | Check Actions tab for errors. Verify `NOTION_TOKEN` in Secrets |
| **Dashboard shows "Fetch failed"** | Hard refresh browser (Cmd+Shift+R on Mac). Check console (F12) |
| **NOTION_TOKEN not found error** | Verify secret is set correctly in repo Settings → Secrets |
| **"Access Denied to Notion" error** | Make sure integration is added to your Notion page (... → Connections) |
